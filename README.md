# LED Photo Lamp - Product Details UI

A pixel-perfect, responsive frontend recreation of a product details page using HTML5, CSS3, and Vanilla JavaScript.

## Features
- **Semantic HTML5** structure with BEM-inspired class naming.
- **Vanilla CSS3** with custom variables, smooth transitions, and hover effects.
- **Responsive Layout** adapting perfectly across mobile, tablet, and desktop.
- **Lucide Icons** integrated dynamically via JavaScript.
- **Interactive Image Gallery** with thumbnail swapping.
- **Photo Upload** with Drag & Drop support and simulated preview.
- **Customization Options** (Shape, Color, Base) with active state toggling.
- **Quantity Selector** with logic.
- **Toast Notifications** for realistic user feedback.

## Setup Instructions

1. Clone or download the repository.
2. Ensure you have the following folder structure:
   ```
   project/
   ├── index.html
   ├── css/
   │   └── style.css
   ├── js/
   │   └── script.js
   ├── assets/
   │   └── images/
   │       ├── main-lamp.jpg
   │       ├── thumb1.jpg
   │       ├── thumb2.jpg
   │       ├── thumb3.jpg
   │       ├── thumb4.jpg
   │       ├── example-photo.jpg
   │       ├── base-wood.jpg
   │       └── base-white.jpg
   ```
3. Open `index.html` in any modern web browser or run it through a local server (e.g. VS Code Live Server).

## Assets
The placeholder images in `assets/images/` can be freely swapped with your actual product photography. Just ensure the filenames match the references in `index.html`.
