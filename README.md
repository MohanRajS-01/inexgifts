# ShopiKart My Orders Portal - React (Vite)

This is the React version of the ShopiKart My Orders portal, converted from the completed vanilla implementation.

## Folder Structure

Follows the teammate's exact layout:
* `public/assets/` - Images and icon assets.
* `src/components/` - Highly modular functional components.
* `src/pages/` - Main page view (MyOrders.jsx).
* `src/hooks/` - Custom React hooks for states (useOrders.js).
* `src/data/` - Static order datasets.
* `src/utils/` - Shared helper utilities.
* `src/css/` - Plain CSS stylesheets.
* `src/App.jsx` - Root Component.
* `src/main.jsx` - Entry DOM mounting.

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```
2. Run development server:
   ```bash
   npm run dev
   ```
3. Build production bundle:
   ```bash
   npm run build
   ```
